# GS1---POO-Plataforma-de-Monitoramento-Espacial
Este repositório contém o Sistema de Monitoramento Espacial, desenvolvido como parte da disciplina de Programação Orientada a Objetos (POO). O projeto simula uma missão espacial, permitindo acompanhar sensores, controlar diferentes tipos de propulsão e gerenciar dados da missão em tempo real.
